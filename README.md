EMI Calculator :

A simple and responsive EMI (Equated Monthly Installment) Calculator built using HTML, CSS, and JavaScript.

This project allows users to calculate monthly loan EMI based on:
Loan Amount
Loan Duration (Years)
Interest Rate (% per year)

Features :

Clean and beginner-friendly UI
Background image design
Instant EMI calculation
Input validation
Supports zero interest rate calculation
Fully client-side (No backend required)

Technologies Used:

HTML5
CSS3
JavaScript

Future Improvements :

Add loan summary section
Add amortization schedule
Dark mode support
Mobile responsive enhancements
Export EMI report as PDF
